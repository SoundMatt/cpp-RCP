# Security Policy — cpp-RCP

## Supported versions

| Version | Supported |
|---|---|
| main | Yes |

## Reporting a vulnerability

Report security vulnerabilities by opening a **private** GitHub Security Advisory:
<https://github.com/SoundMatt/cpp-RCP/security/advisories/new>

Do **not** open a public issue for security vulnerabilities.

We aim to acknowledge reports within 2 business days and provide a fix or mitigation plan within 14 days.

## Security standard

cpp-RCP targets IEC 62443 SL-2. See `.fusa-iec62443.json` and `INCIDENT-RESPONSE.md`.
